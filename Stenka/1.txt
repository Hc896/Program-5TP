Odp1:
NodeJS
Środowisko uruchomieniowe do wykonywania kodu JavaScript po stronie serwera. Służy do budowania aplikacji backendowych, serwerów API i narzędzi CLI.
NPM
Node Package Manager – menedżer pakietów dla JavaScript. Służy do instalowania, zarządzania i udostępniania bibliotek oraz zależności w projektach Node.js i frontendowych.
React
Biblioteka JavaScript do tworzenia interfejsów użytkownika (UI). Służy głównie do budowania dynamicznych aplikacji webowych, zwłaszcza single-page applications (SPA), z komponentami wielokrotnego użytku.
Vite
Narzędzie do budowania i serwera deweloperskiego dla nowoczesnych projektów webowych. Służy do szybkiego developmentu i bundlowania aplikacji (np. z Reactem), oferując szybsze ładowanie niż tradycyjne narzędzia jak Webpack.
